User Manual 

This is the manual explaining how to use the Eastern Medical Health Records System.
### Running the Program
Import the `.java` files into eclipse.
### Logging In
At the start of the program, you will be presented with a login screen. If logging in for the first time, you may use the default login credentials:
```
username: Admin
password: Admin123
```
This will take you to the default Administrator account, which gives you access to the entire system. You will then be presented with the main menu from which you can select the options.
If the login credentials fail you will be prompted to renter the password again.
## Main Menu
This is the screen presented after logging in. On this screen you will be presented with several options. To select one of the options simply type the number corresponding to the options. If you enter an invalid option, such as a value out of range, or a letter you will be prompted to re-enter.
The following are the options and how they work:

1. **Add Patient** – Enter the patient name, ID, contact number and Next of Kin Information. The Patient will then be added to the system. To access the option you need to be logged in as an Administrator.
2. **View Patient** – Allows you to view all the patients added to the system.
3. **Update Patient** – Enter the ID number of the patient whose information you want to update. You will then be allowed to change the contact number and Next of Kin information. The information is then updated in the system.
4. **Delete Patient** – Enter the ID of the patient you want to delete and it will be removed from the system.
5. **Add User** – Allows you to add a new user to the system. Enter their full name, username and password. Make sure the password is at least 8 characters long and contains letters and numbers or the system will prompt you to re-enter.
6. **View Single User** – Enter the username and you will be able to view all the information about the user.
7. **View All Users** – Allows you to view all the users added to the system.
8. **Update User** – Enter the name of the user to update, and the program allows you to change the information. If the username is invalid you will see “Failed to update user”.
9. **Delete User** – Enter the username of the user to delete. If the username doesn’t exist it will say “failed to delete user”.
10. **View my profile** – view the information of the current user who is logged into the system.
11. **Schedule Appointment** – Schedule appointment, enter the staff and patient ID and the date/time  of the appointment.
12. **View Appointment** – see a list of all appointments.
